# Practice 01: The Golden Workflow

**Explore → Plan → Implement → Commit**

Experience the core Claude Code workflow that is repeatedly emphasized across all resources.

## What You'll Learn

1. **Explore**: How to explore the codebase in Plan Mode
2. **Plan**: How to create a change plan and get it approved
3. **Implement**: How to modify code according to the plan
4. **Commit**: How to commit and organize your changes

## Prerequisites

- Claude Code installed ([Getting Started](../../docs/getting-started.md))
- This repository cloned

## Time Required

20-30 minutes

## Why This Matters

> "Don't jump straight into coding. If you verify your approach in Plan Mode before implementing, you'll save ~20 minutes of post-hoc debugging time."
> — Claude Code Master Guide 2026

Most beginners immediately say "build this for me," but **exploring first and creating a plan** leads to:
- Claude following existing code patterns
- Preventing large amounts of code from being generated in the wrong direction
- Efficient use of context
