# Practice 01: 황금 워크플로우

**Explore → Plan → Implement → Commit**

모든 자료에서 반복적으로 강조하는 Claude Code의 핵심 워크플로우를 직접 체험합니다.

## 배울 것

1. **Explore**: Plan Mode에서 코드베이스를 탐색하는 방법
2. **Plan**: 변경 계획을 세우고 승인받는 방법
3. **Implement**: 계획에 따라 코드를 수정하는 방법
4. **Commit**: 변경사항을 커밋하고 정리하는 방법

## 사전 요구

- Claude Code 설치 완료 ([Getting Started](../../docs/getting-started.md))
- 이 레포를 클론한 상태

## 소요 시간

20~30분

## 왜 중요한가

> "바로 코딩하지 마세요. Plan Mode로 접근 방식을 확인한 후 구현하면 사후 디버깅 시간을 ~20분 절약합니다."
> — Claude Code 마스터 가이드 2026

대부분의 초보자가 바로 "이거 만들어줘"라고 하지만, **먼저 탐색하고 계획**을 세우면:
- Claude가 기존 코드 패턴을 따름
- 잘못된 방향으로 대량의 코드가 생성되는 것을 방지
- 컨텍스트를 효율적으로 사용
